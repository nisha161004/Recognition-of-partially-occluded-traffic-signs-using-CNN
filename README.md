# Recognition of Partially Occluded Traffic Signs using CNN

This project uses Convolutional Neural Network (CNN) model to recognize traffic signs even when they are partially occluded.

## Project Overview
Traffic sign recognition is an important component of intelligent transportation systems and autonomous driving. This project focuses on identifying road signs when parts of the sign are hidden or blocked.

## Features
- CNN-based image classification
- Handles partially occluded traffic sign images
- Displays occluded input image along with a clear reference image for comparison
- Predicts the traffic sign category

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

## Model Workflow
1. Load and preprocess traffic sign dataset
2. Train CNN model for image classification
3. Evaluate model performance
4. Predict traffic sign from occluded images
5. Display prediction results with comparison images

## Output
The system predicts the traffic sign and shows the occluded input image alongside the clear reference image for visual comparison.
